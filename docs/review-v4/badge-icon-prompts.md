# BabyMon Badge Icon Generation Prompts

Complete production-ready prompts for all 86+ BabyMon badges.
Source: `apps/api/src/badges/badges.service.ts` — `BADGE_DEFINITIONS` + `checkAndAwardBadges`.

---

## Art Direction Guidelines

- **Shape**: Perfectly circular badge, 512×512 output
- **Style**: Flat-vector, soft rounded shapes, gamified achievement, cute illustration (no photorealism)
- **Colors**: Pastel-forward, warm, nurturing. Bronze (copper), Silver (steely-blue), Gold (amber/honey), Diamond (cosmic/prismatic)
- **Mood**: Celebratory yet tender — celebrates baby's journey and parent's dedication
- **Production**: `--ar 1:1 --style cute --no text,photorealistic,scary,sharp edges,clinical`
