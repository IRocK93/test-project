# BabyMon Firebase Configuration
# ==============================
#
# Firebase is used for push notifications and analytics.
# To configure Firebase for your project:
#
# 1. Go to https://console.firebase.google.com/
# 2. Create a new project (or use existing)
# 3. Add an Android app with package name: YOUR_APP_ID
# 4. Download google-services.json and place it at:
#    apps/mobile/android/app/google-services.json
# 5. Add an iOS app with bundle ID: YOUR_BUNDLE_ID
# 6. Download GoogleService-Info.plist and place it at:
#    apps/mobile/ios/Runner/GoogleService-Info.plist
# 7. Enable Cloud Messaging for push notifications
# 8. Enable Analytics if desired
#
# Android build.gradle already has the Google Services plugin.
# iOS Podfile already includes Firebase dependencies.
# The firebase_core and firebase_messaging packages are in pubspec.yaml.
#
# ⚠️ IMPORTANT: google-services.json and GoogleService-Info.plist
# contain API keys and should NOT be committed to version control.
# Add them to .gitignore!
